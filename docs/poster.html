Why Registry Config Engine?

Transaction Rollback

Every change is logged. Restore previous values instantly during testing and development.

All User Profiles

Apply to existing users, new users via DefaultUser hive, and supports both AD and Entra ID SIDs.

WhatIf Validation

Preview all changes before applying. Built-in -WhatIf support for safe testing.

Smart Comparison

Beyond equals: GreaterThan, Contains, StartsWith, Exists, NotExists for flexible detection.

Dynamic Variables

Use {{DATE}}, {{COMPUTERNAME}}, {{USERNAME}} and more for runtime value expansion.

Self-Contained Packages

New-IntunePackage.ps1 embeds config + engine into standalone detect/remediate scripts.
